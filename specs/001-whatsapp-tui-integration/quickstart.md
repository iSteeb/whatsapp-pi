# Quickstart: WhatsApp TUI Integration

## Setup
1. Open Pi TUI.
2. Type `/whatsapp`.
3. Select **Login**.
4. Scan the QR code displayed in your terminal with your phone.

## Enabling the Agent
1. Type `/whatsapp`.
2. Select **Allow Numbers**.
3. Add your phone number (e.g., `+5511999999999`).
4. Select **Connect**.

The Agent will now answer messages coming from the added number!

## Security
- To stop the Agent without logging out, select **Disconnect**.
- To remove all credentials, select **Logoff**.
